Pitch-Class Set Bot

This page is a reference for my Pitch-Class Set Bot. All of the code the bot uses to analyze the sets and rows can be found on my GitHub. What follows is an attempt to explain the content of the of the tweets in layman's terms.

Set


A pitch class set is simply an unordered collection of notes. Strictly speaking, the sets {C4, E4, G4} and {C5, E5, G5} (where C4 = middle C) are different because the notes are in different octaves. With 88 notes on the piano, even if we restrict ourselves to only two-note sets, there are still 3,828 unique sets! And this doesn't take into account all the possible notes extending infinitely beyond the range of the piano in both directions! We can reduce the number of possible sets quite a bit by invoking the notion of octave equivalence.
