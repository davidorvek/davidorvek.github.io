Pitch-Class Set Bot

This page is a reference for the Pitch-Class Set Bot. This bot makes use of my Python library of pitch-class set tools that can be found on my GitHub. Each day, the bot creates one random pitch-class set (anywhere from 2 to 10 pitch classes) and one random twelve-tone row and tweets various facts about that set or tone row.
